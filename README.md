## 本コンテンツについて

本リポジトリは、**AWS CloudFormationを用いたAWS環境構築を学習するための教育用コンテンツ**です。

CloudFormationの基本的な考え方やテンプレートの書き方を学び、実務において「CloudFormation を使うことになった際に最低限対応できる状態」になることを目標としています。

## 対象者

以下の方々を対象としています。

- CloudFormationを初めて学ぶ方
- AWSの基本的なサービス（VPC、EC2、IAM など）について最低限の知識がある方
- YAMLの基本的な記法を理解している、または学習意欲がある方
- 案件でCloudFormationを使うことになり、急遽学習が必要になった方

## 学習の進め方

基本的な学習の進め方は以下としていますが、自分に合うようにカスタマイズしていていただいても問題ございません。

1. `chapter00/` から順番に各章を読み進める
2. `templates/examples/` 内のサンプルテンプレートを実際に動かしてみる
3. `chapter09/README.md` の演習問題に取り組む
